# bilna-test
Bilna Test

## Installation 
```ssh 
git clone https://github.com/agusfantasy/bilna-test.git
npm install
```

## Usage
```ssh
node index.js [options] [text]

Options:
-p : determine is Palindrom
-a : determine is Anagram
```

Example:
```ssh
node index.js -p malam

node index.js -a bali bila
```

## Depedency (global)
- mocha

## Test
mocha test
